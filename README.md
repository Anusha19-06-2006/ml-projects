Heart Disease Detection using Machine Learning

1.About the Project

This project predicts whether a person has heart disease based on medical information such as age, chest pain type, cholesterol level, blood pressure, and other health parameters.

The goal is to analyze the data and build a machine learning model that can accurately detect heart disease.

2. Dataset Features

The dataset includes:

Age

Sex

Chest Pain Type

Resting Blood Pressure

Cholesterol

Fasting Blood Sugar

Resting ECG Results

Maximum Heart Rate Achieved

Exercise Induced Angina

ST Depression

Slope of ST Segment

Number of Major Vessels

Thallium Test Result

3.Target:

0 → No Heart Disease

1 → Heart Disease Present
